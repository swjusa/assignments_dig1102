Operators

Operators = is used to assign values + is used to add values

txt1="What a very"; txt2="nice day"; txt3=txt1+txt2; Result: What a very nice day

Comparison and Logical Operators are used to text for true or false

x==8 (equal to) x==="5" (exactly equal to (equal to value and equal to type)) x!=8 (not equal) x!=="5" (not equal (because of different value or different type)) x>8 (greater than) x<8 (less than)

Logical Operators (x<10 && y >1) is true (x==5 || y==5 is false !(x==y) is true
